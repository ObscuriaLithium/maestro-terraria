# Features

### Extensive Music Coverage
The pack features unique music across the Overworld, The Nether, and The End, with additional themes for biomes, villages, dungeons, boss fights, raids, weather events, and multiple world depth levels. Each situation transitions naturally into the next, ensuring consistent atmosphere without abrupt cuts or repetition.

### Advanced Usage Example
Maestro: Terraria can also serve as a reference implementation for developing your own music packs with Maestro. The project demonstrates advanced use of track priorities, layered orchestration and composite conditions working together in a real production setup.

# Good to Know

<br>

### Requires Maestro
Maestro: Overture is a music resource pack and requires the
[Maestro](https://modrinth.com/mod/maestro-music) mod to function. Without Maestro, the pack will not affect music playback.
